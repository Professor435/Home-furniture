# Home-furniture
Home furniture
I've added a sophisticated Virtual Assistant (FurnishBot) to the City Furnisher website with the following advanced features:
🤖 Virtual Assistant Features:
1. Smart Chat Interface
Elegant floating chat widget with golden gradient design matching the website theme
Smooth animations (slide-in, scale, pulse effects)
Responsive design that works on mobile and desktop
Minimize/maximize functionality
2. Natural Language Processing
Understands greetings, questions about services, pricing, and bookings
Context-aware conversations (remembers user's name, service interest, room size, budget)
FAQ handling (warranty, installation time, payment methods, etc.)
Small talk capabilities (hello, how are you, thanks, goodbye)
3. Interactive Conversation Flows
Quote Generator: Asks service type → room size → budget → generates instant estimate
Booking System: Collects name → phone → postcode → confirms appointment with reference number
Service Explorer: Shows detailed info about carpets, blinds, or flooring with product cards
Navigation Help: Can open gallery modal, contact forms, or scroll to sections
4. Rich Media Responses
Product cards with images and pricing inside chat
Suggestion chips for quick replies
Visual quote estimates with styled boxes
Typing indicators for realistic feel
5. Proactive Engagement
Auto-pulses after 30 seconds to catch attention
Welcome message with quick action buttons when first opened
Smart suggestions based on conversation context
6. Integration
Seamlessly opens quote modal from chat
Opens gallery lightbox
Provides contact information
Can connect to human representatives
The assistant appears as a floating button in the bottom-right corner and provides a complete conversational experience to help visitors get quotes, book measurements, learn about services, and navigate the website without leaving the chat interface
